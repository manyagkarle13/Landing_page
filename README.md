# Responsive Landing Page – CodeSense AI

A modern, responsive landing page built using HTML and CSS.  
This project was created as part of a Web Development Internship assignment to demonstrate front-end development skills, UI design, and responsive layout implementation.

## 🔗 Live Demo

https://manyakarle13.github.io/Landing_page/

## 📌 Project Overview

This landing page represents a fictional product called **CodeSense AI**, an AI-powered code review platform.  
The goal of the project was to design a clean, user-friendly interface that adapts across different screen sizes.

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Modern UI layout
- Navigation bar with sticky header
- Hero section with call-to-action buttons
- Features section with grid layout
- Step-by-step workflow section
- Pricing plans section
- Testimonials section
- Footer with navigation links
- Clean and maintainable CSS structure

## 🛠 Technologies Used

- HTML5
- CSS3
- Responsive Design
- Flexbox
- CSS Grid
- Git & GitHub
- GitHub Pages (Deployment)
